# Skill: Perform Frequency Response Analysis

## Purpose

Acquire and analyze frequency response data to characterize system dynamics and identify resonances.

## Inputs

- Signal generator or sweep excitation input.
- Data acquisition system for measuring input and output signals.
- Signal processing tools (FFT, spectrum analyzer).

## Procedure

1. Excite the system with a swept sine or multi‑sine input over the frequency range of interest.
2. Measure input and output; compute the transfer function using FFT or cross‑spectral methods.
3. Plot magnitude and phase versus frequency; identify resonant peaks and bandwidth.
4. Extract modal parameters or design filters and controllers based on the FRF.

## Validation

Validate FRF results by comparing predicted response with measured time‑domain data; ensure coherence and repeatability.
