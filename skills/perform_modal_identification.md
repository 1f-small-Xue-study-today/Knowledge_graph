# Skill: Perform Modal Identification

## Purpose

Determine modal parameters (natural frequencies, damping ratios, mode shapes) from frequency response data.

## Inputs

- Frequency response functions or impulse response data.
- Signal processing and curve‑fitting tools.

## Procedure

1. Select candidate modes by identifying peaks in the FRF.
2. Apply curve fitting or subspace identification to extract modal parameters.
3. Estimate damping ratios and modal participation factors.
4. Validate identified modes using residual checks and by reconstructing responses.

## Validation

Use identified modal models to simulate system response and compare with experimental data; ensure error metrics are within acceptable limits.
