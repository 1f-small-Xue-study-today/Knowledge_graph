# Plan Folder

This folder is permanent and should not be deleted.

Use it to keep short-term and long-term study plans, current weekly tasks, and future implementation plans. Completed plans can be archived or summarized, but the folder itself remains part of the knowledge graph.

## Structure

Plans are grouped by year first. `2026/` is the first year folder.

- `<year>/short_term/`: one-week plans and other near-term execution plans.
- `<year>/long_term/`: month-scale plans and broader roadmaps.

## Naming

Short-term plan files should use year plus ISO week number:

```text
<year>/short_term/YYYY-Www.md
```

Example:

```text
2026/short_term/2026-W19.md
```

Long-term plan files should use year plus month:

```text
<year>/long_term/YYYY-MM.md
```

Example:

```text
2026/long_term/2026-05.md
```
