# Demo Workspace

This folder contains runnable MATLAB demos for study topics.

## Demos

- `ransac/modal_fit_sweep/`: RANSAC modal fitting from sweep data.
- `ransac/homography_implement/`: RANSAC homography estimation in MATLAB.
- `l_bfgs/`: L-BFGS optimization on the Rosenbrock objective.

Run the main script under each topic's `scripts/` folder.
