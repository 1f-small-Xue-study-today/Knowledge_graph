# Control Theory

## Overview

Control theory studies how to manipulate the input of a dynamical system to achieve a desired output behaviour.  Classical control uses transfer functions and frequency domain techniques such as Bode and root locus.  Modern control represents systems in state‑space form and employs matrix algebra and optimal control.

## Key Areas

- Feedback stability and robustness
- Time domain performance (rise time, overshoot, steady‑state error)
- Frequency domain analysis (phase and gain margins)
- State‑space representation and controllability/observability
- Optimal control and estimation (LQR, Kalman filter)
