# Signal Processing Basics

## Overview

Signal processing involves analysis and manipulation of signals to extract useful information.  In control systems, it is used for filtering, estimation and modulation.

## Concepts

- Sampling and aliasing
- Fourier transform and spectral analysis
- Digital filters (FIR, IIR)
- Noise filtering (low‑pass, band‑pass)
- Time‑domain versus frequency‑domain representations
