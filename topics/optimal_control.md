# Optimal Control

## Overview

Optimal control seeks to find a control law that minimises a cost function subject to system dynamics.  Problems include LQR, minimum‑time and minimum‑energy control, and optimal trajectories.

## Elements

- **Cost function:** Defines performance measure (e.g., quadratic cost, fuel consumption).
- **Constraints:** System dynamics and state/input limits.
- **Solution methods:** Pontryagin’s Minimum Principle, dynamic programming, Riccati equation.
