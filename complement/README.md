# Complement Directory

This folder stores weekly notes on bugs, new issues or discoveries encountered during study and development.  Each week, create a file named according to the week (e.g., `week_of_2026-03-09.md`) and record:

* Unexpected bugs or obstacles faced during the plan.  
* New topics or techniques discovered that are not yet represented in the knowledge graph.  
* Short summaries explaining the cause of each issue and how it was resolved (if applicable).

At the end of each week, abstract the content of the complement notes to update other folders and `graph.yaml` as needed.
